Welcome to DataScape Visualizer!

The idea behind DataScape Visualizer is offer visualizations
by analyzing data. The data collected can be uploaded by .csv
format or by manually entering the data from the the Visualizer
section. This dynamic website has the ability to give the user
to create an account and save all data to a database.

The visualizations will only be available in 2D style.
Some features are the following: bar chart, scatter plot,
bubble chart, and line chart. In the upcoming months,
more visualizations will become available.
